# Event Sourcing cho dự án: phạm vi tài liệu PMP + bộ input cho AI Agent

Tài liệu này tóm tắt:
1. **Những process trong 49 process PMP cần cập nhật** khi triển khai Event Sourcing.
2. **Bộ file Markdown nên có** để AI agent có thể hỗ trợ phân tích, thiết kế, triển khai, kiểm thử, và vận hành.

## Kết luận nhanh
- Event Sourcing tác động mạnh đến các nhóm process: **Scope, Schedule, Cost, Quality, Resource, Communications, Risk, Procurement, Stakeholder**, và đặc biệt là **Planning + Monitoring & Controlling**.
- Không phải 49 process đều cần chỉnh sâu như nhau; một số process chỉ cần “rà soát và ghi nhận không đổi”.

## Cấu trúc thư mục `event-sourcing`
- `01-pmp-49-process-update-matrix.md`: ma trận 49 process và mức độ cần update.
- `02-ai-agent-input-checklist.md`: checklist file đầu vào AI.
- `03-12`: bộ template nội dung cốt lõi cho Event Sourcing.
